8 6 ລາຍການ

👈 ຄລິກໃສ່ໝາຍເລກສີຟ້າເທິງຮູບ ເພື່ອເລີ່ມຮຽນຮູ້ແຕ່ລະສ່ວນ.

ອ່ານຂັ້ນຕອນ ແລ້ວ ລອງເຮັດເອງ ໃນກ່ອງ Sandbox ດ້ານລຸ່ມ — ກົດປຸ່ມ ຫຼື ໃຊ້ປຸ່ມລັດກໍ່ໄດ້.

ສ້າງໂຟນເດີໃໝ່

  • 1ໄປຫາບ່ອນທີ່ຢາກສ້າງ (ເຊັ່ນ Documents).
  • 2ກົດປຸ່ມ New (+) ເທິງແຖບຄຳສັ່ງ ແລ້ວເລືອກ Folder — ຫຼື ຄລິກຂວາໃນບ່ອນວ່າງ › New › Folder.
  • 3ພິມຊື່ໂຟນເດີ ແລ້ວກົດ Enter.

ປຸ່ມລັດ: Ctrl+Shift+N

ປ່ຽນຊື່ໄຟລ໌ / ໂຟນເດີ

  • 1ຄລິກໃສ່ໄຟລ໌ ຫຼື ໂຟນເດີ ໜຶ່ງເທື່ອ ເພື່ອເລືອກ.
  • 2ກົດປຸ່ມ F2 (ຫຼື ກົດປຸ່ມ Rename / ຄລິກຂວາ › Rename).
  • 3ພິມຊື່ໃໝ່ ແລ້ວກົດ Enter.

⚠️ ໄຟລ໌ບໍ່ຄວນລຶບສ່ວນນາມສະກຸນ ເຊັ່ນ .docx ອອກ.

ກັອບປີ້ ແລະ ວາງ (Copy & Paste)

  • 1ຄລິກເລືອກໄຟລ໌ ຫຼື ໂຟນເດີ.
  • 2ກົດ Ctrl+C ເພື່ອກັອບປີ້ — ຫຼື Ctrl+X ເພື່ອ ຕັດ/ຍ້າຍ.
  • 3ໄປຫາໂຟນເດີປາຍທາງ ແລ້ວກົດ Ctrl+V ເພື່ອວາງ.

ຄວາມຕ່າງ: Copy = ມີ 2 ສຳເນົາ  |  Cut = ຍ້າຍໄປບ່ອນໃໝ່ (ບ່ອນເກົ່າຫາຍ).

Sandbox — ລອງເຮັດເອງ

ເລືອກໄຟລ໌/ໂຟນເດີກ່ອນ ແລ້ວກົດປຸ່ມ. ໃຊ້ໄດ້ໝົດທັງ ສ້າງໃໝ່, ປ່ຽນຊື່, ກັອບປີ້, ວາງ, ລຶບ.

ແບບເຝິກຫັດ 1 — ລາກໝາຍເລກໃສ່ຊື່ສ່ວນໃຫ້ຖືກ

ລາກໝາຍເລກຈາກກ່ອງ ໄປວາງໃສ່ໜ້າຊື່ສ່ວນທີ່ຖືກຕ້ອງ — ຫຼື ກົດໝາຍເລກ ແລ້ວກົດໃສ່ຊ່ອງວ່າງກໍ່ໄດ້.

ແບບເຝິກຫັດ 2 — ຄຳຖາມເລືອກຕອບ

ກົດເລືອກຄຳຕອບທີ່ຖືກຕ້ອງ. ຖືກ = ສີຂຽວ, ຜິດ = ສີແດງ.

ສ້າງເພື່ອການສອນວິຊາພື້ນຖານຄອມພິວເຕີ — Windows 11 File Explorer
